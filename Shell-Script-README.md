# 🐚 Shell Scripting Journey — Bash for DevOps

> A hands-on Shell Scripting learning repository where I practice Bash scripting concepts day by day — from basics to real-world DevOps automation scripts.

---

## 👨‍💻 About This Repo

This repository documents my **day-by-day Shell Scripting practice** as part of my DevOps learning journey. Each folder contains scripts, notes, and hands-on assignments from that day's learning session.

- 🎯 **Goal:** Master Shell Scripting for DevOps Automation
- 🗓️ **Structure:** Day-wise organized folders
- 💡 **Approach:** Hands-on practice + real DevOps examples

---

## 📁 Folder Structure

```
Shell-Script/
├── Day-1/    # Bash Basics — Variables, User Input, System Variables
├── Day-2/    # Conditional Statements — if, if-else, File Checks
├── Day-3/    # Loops — for, while, Infinite Loop, Monitoring Scripts
```

## 📚 Topics Covered

### 📅 Day 1 — Introduction to Bash Scripting
- What is Shell Scripting & why it matters in DevOps
- Creating and running your first script (`hello.sh`)
- Variables — storing and using data
- Taking user input with `read`
- Built-in system variables — `$USER`, `$HOME`, `$PWD`
- Real DevOps example: Deployment Notification Script
- Execute permissions — `chmod +x`

### 📅 Day 2 — Conditional Statements
- `if`, `if-else` statements
- Comparison operators — `-eq`, `-ne`, `-gt`, `-lt`, `-ge`, `-le`
- File existence check — `-f`, `-d`, `-r`, `-w`, `-x`
- Nested if statements
- Real DevOps example: Nginx Service Status Checker

### 📅 Day 3 — Loops & Automation
- `for` loop — list-based and range-based (`{1..5}`)
- `while` loop — condition-based repetition
- Infinite loop with `while true` + `sleep`
- Real DevOps example: Server Monitoring Script (memory + date)
- Backup automation — copy all `.txt` files automatically

### 📅 Day 4 — Functions & Reusable Scripts
- Writing and calling functions
- Passing arguments to scripts
- Return values
- Reusable deployment & backup utility scripts

## ▶️ How to Run These Scripts

```bash

git clone https://github.com/Sapra007/Shell-Script.git

# Navigate to any day folder
cd Shell-Script/Day-1

# Give execute permission
chmod +x script.sh

# Run the script
./script.sh
```
